WebXR Hand Tracking Playground

A collection of interactive WebXR demos that explore hand tracking, VR interactions, physics, water/visual effects, and VRM character animation using Three.js.

Browse demos View on GitHub
About this project

Built with Three.js, WebXR, TypeScript, and Webpack. Best experienced on a headset with hand tracking (e.g., Meta Quest), but most scenes also open on desktop/mobile browsers.

Hand tracking VRM animation Physics Post‑processing Water & ocean
